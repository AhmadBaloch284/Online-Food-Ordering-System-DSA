# 🍔 Online Food Ordering System (DSA Project)

---

## 📖 Overview
This project is an **Online Food Ordering System** built using core **Data Structures concepts**.  
It allows customers to register, browse menus, place orders, and track order history efficiently.

The system uses:
- **BST / AVL Trees** for menu management  
- **Queue** for order processing  
- **Stack** for order history  
- **Linked List** for customer records  

---

## 🚀 Features
- 🧾 Customer registration & login  
- 🍽️ Menu management (add, search, delete, update)  
- 🛒 Order placement and processing  
- ⏳ FIFO-based order handling using Queue  
- 🔄 Undo/Redo using Stack  
- 📦 Stock and inventory management  
- 📊 Customer order history tracking  

---

## 🧠 Data Structures Used

### 🌳 BST / AVL Tree
- Efficient menu item storage  
- Fast search, insert, delete operations  
- Maintains sorted order  

### 📥 Queue (FIFO)
- Processes orders in **First-Come-First-Serve** manner  

### 📚 Stack (LIFO)
- Stores order history  
- Supports undo/redo operations  

### 🔗 Linked List
- Manages customer records dynamically

## 🔄 Workflow
1. Customer registers/logs in  
2. Browses menu  
3. Places order  
4. Order added to Queue  
5. System processes order (FIFO)  
6. Stock updates automatically  
7. Order saved in history (Stack)
